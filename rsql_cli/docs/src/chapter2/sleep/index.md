## sleep

### Usage

```text
.sleep <seconds>
```

### Description

The sleep command pauses the CLI for the specified number of seconds. This command is can be used in scripts
to simulate delayed user interaction with a data source.

### Examples

Sleep for 1 second:

```text
.sleep
```

Sleep for 3 seconds:

```text
.sleep 3
```

Sleep for .5 seconds:

```text
.sleep .5
```
